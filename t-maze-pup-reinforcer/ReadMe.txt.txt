Files to run T-maze sound playback pseudorandomizing sound output between two channels. Use TDT OpenEx running on RP2.1 (Ctrl) and RX6 (Stim) System 3 units. Use TDT OpenEx OpenProject to open natAppTF_amnoise.wsp file. Uses TDT OpenController and OpenWorkbench.

For questions, contact robert.liu@emory.edu
